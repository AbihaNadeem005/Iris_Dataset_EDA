# Task 1: Exploring and Visualizing the Iris Dataset

## Project Overview  
The goal of this task is to explore, analyze, and visualize the Iris dataset to understand data patterns, distributions, and relationships between features.

## Objective
- Load and inspect the Iris dataset  
- Perform exploratory data analysis (EDA)  
- Visualize data distributions and relationships  
- Identify patterns between different flower species  

## Dataset Used
- Name: Iris Dataset  
- Source: Built-in dataset from Seaborn (sns.load_dataset('iris'))  

Description:  
The dataset contains 150 samples of iris flowers with 4 features:
- Sepal length  
- Sepal width  
- Petal length  
- Petal width  

And 3 species:
- Setosa  
- Versicolor  
- Virginica  

## Tools & Libraries Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn  

## Tasks Performed
- Loaded dataset using Pandas/Seaborn  
- Inspected data using .head(), .shape(), .info(), .describe()  
- Performed exploratory data analysis (EDA)  
- Created visualizations:
  - Scatter Plot (feature relationships)  
  - Histograms (data distribution)  
  - Box Plot (outlier detection)  

## Models Applied
No machine learning models were applied in this task.  
This task focused only on data exploration and visualization.

## Key Results & Findings
- The dataset contains 150 rows and 5 columns  
- No missing values were found  
- Petal length and petal width are the most important features for distinguishing species  
- Setosa species is clearly separable from the other two species  
- Data visualization helped in understanding feature distributions and relationships
